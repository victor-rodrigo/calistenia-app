# calistenia-app

App Android para gerenciar meus treinos de calistenia. Uso pessoal, funciona
totalmente offline — os dados ficam no próprio aparelho.

Permite montar fichas de treino (dias, exercícios, séries, reps/tempo e descanso),
registrar as sessões e manter o histórico, acompanhar a progressão de habilidades
(muscle-up, front lever, etc.) e visualizar a evolução em gráficos, com timer de
descanso durante o treino.

## Stack

- **Flutter** (Dart) + Material 3
- **Drift** (SQLite) para persistência local
- **Riverpod** para estado
- **go_router** para navegação
